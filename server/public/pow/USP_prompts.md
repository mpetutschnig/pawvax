# Vetzsucht — USP-Deck (Neukonzept)

**Konzept:** Letztes Mal wurden wir unsanft nach dem USP gefragt. Dieses Deck
ist die Antwort — als Konter, nicht als Aufzählung. Roter Faden: die fiese
Frage selbst. Schlag auf Schlag, ein Bild pro Folie, Text als Overlay über der
schwarzen Bildmitte.

**Bogen:** Frage → Schmerz → Konkurrenz verspotten → Wende → 3 USP-Schläge →
Punch → Name → Callback.

**Bildstil (für ALLE Prompts):** Cinematic dark photography, ein Tier am
Bildrand, pechschwarze, leere Mitte für Text-Overlay, dramatisches Licht,
hoher Kontrast, moody. `--ar 16:9 --style raw`

Hinweis: `slide-XX.jpg` = Dateiname, den die Folie erwartet. Overlay-Text wird
per CSS über das Bild gelegt — nicht ins Bild rendern lassen.
