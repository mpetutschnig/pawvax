Papier. / Verloren.
Problem-Hook. Jeder kennt das — Impfpass vergessen, beim Tierarzt gesucht, nicht gefunden. Emotionaler Einstieg, keine Erklärung nötig.

Nie wieder.
Payoff zum Problem. Kurz. Wirkt wie eine Aussage, kein Versprechen.

Smartphone. / An Tag. Fertig.
Sofort in die Lösung. NFC als primärer USP — simpelste denkbare Interaktion. Drei Worte beschreiben die komplette User Journey.

QR. NFC. Link. / Drei Wege. Ein Ziel.
Zeigt Flexibilität — nicht nur NFC, sondern auch QR-Code und direkter Link. "Ein Ziel" signalisiert: egal wie, das Ergebnis ist dasselbe. Breite Kompatibilität ohne technisches Brimborium.

Ohne Login. / Sofort sichtbar.
Kritischer Differentiator. Tierarzt braucht keinen Account. Keine Hürde. Daten sind sofort da — das ist konkreter Mehrwert für den Fachmann.

Tierarzt scannt. / Alles da.
Persona wechsel — jetzt aus Sicht des Dienstleisters. Workflow komplett abgedeckt ohne Systemwechsel oder Papier.

KI liest. / KI versteht. / KI speichert.
Drei-Zeiler mit Rhythmus. Jede Zeile ist ein Schritt des AI-Prozesses. "Liest" = OCR/Dokumentanalyse, "Versteht" = Extraktion, "Speichert" = Automatisierung. Klingt mächtig ohne technisch zu werden.

Sprechen genügt.
Speech-to-text in zwei Worten. Keine Erklärung. Der Tierarzt diktiert — PAW transkribiert. Minimalste Aussage, maximale Wirkung.

Automatisch übersetzt.
Internationaler USP. Auslandsreise mit Tier, fremdsprachiger Tierarzt — kein Problem. Eine Folie, eine Aussage.

Strukturiert. / Sofort. / Überall.
Drei Adjektive die den Output beschreiben. Keine Rohdaten, sondern organisierte Information — und zwar ohne Wartezeit, auf jedem Gerät. Technisches Versprechen in Nutzersprache.

Ein Scan. / Alles drin.
Zusammenfassung der gesamten Value Proposition. Reduktion auf das Wesentliche — ein einziger Trigger, vollständige Information.

Kein Aufwand. / Null Fehler.
Adressiert den größten Pain Point bei Digitalisierung: "Wird das komplizierter?" Antwort: nein. "Macht das Fehler?" Nein. Direktes Entkräften von Einwänden.

PAW. / eCard for pets.
Brand-Closing. Erst Name, dann Positionierung. Kurz, merkbar, international einsetzbar.
