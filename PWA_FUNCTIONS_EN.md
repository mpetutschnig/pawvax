# PAW on paw.oxs.at - Features for End Users

The PWA on paw.oxs.at is a digital pet passport for pet owners, veterinarians, authorities, and other authorized parties. It is designed to manage animal data, tags, and documents in one place and to share that information in a controlled way.

## What paw.oxs.at offers

### Manage your animals
- Create animals within your own account
- Store name, species, breed, and date of birth
- Edit animal profiles later
- Upload a profile image for an animal
- Search animals and recognize archived animals
- Transfer animals to another account or accept a transfer using a transfer code

### Find animals by tag or scan
- Look up animals by barcode, QR code, or NFC tag
- Assign tags directly to an animal
- Link unknown tags to a new animal immediately
- Open public animal information without logging in if the owner has shared it

### Capture and manage documents
- Store vaccination records, medication documents, and other files for an animal
- View uploaded documents in the animal profile
- Use automated analysis so document contents become easier to read in structured form
- Retry failed analyses
- Find documents more easily by type, date, or content

### Share data selectively
- Define which data should be visible for each animal
- Use different sharing settings for the public view, veterinarians, and authorities
- Generate public sharing links
- Show only the information that has been explicitly approved for sharing

### Use profile and account functions
- Manage your personal user profile
- Submit verification requests for special roles
- Save your own AI or model settings for document analysis
- Export your personal data
- Delete your account

### Role-based access
- Pet owners manage their animals and sharing settings
- Verified veterinarians or authorities can see more shared information where permitted
- Administrators get access to additional management features

## How the app works in daily use

### 1. Register or sign in
After signing in, paw.oxs.at shows your personal animal list, your profile, and all features available for your role.

### 2. Add an animal or find one by scanning
An animal can be created manually or found through tag recognition. If a scanned tag is not known yet, a new animal profile can be created immediately.

### 3. Add and review documents
You can store documents for each animal. The app helps present that information in a way that makes vaccinations and other records easier to review.

### 4. Control sharing
The owner decides which information is visible to whom. This allows public scans to show only basic information, while authorized roles can see more.

### 5. Use the public view
When a sharing link is opened or a tag is scanned, paw.oxs.at shows only the data that the animal owner has released for that specific use case.

## How the paw.oxs.at PWA works

paw.oxs.at is built as a Progressive Web App. That means the application runs directly in the browser, but can still be used almost like a normal app on smartphones and desktop devices.

### PWA characteristics
- Works on smartphones, tablets, and desktop devices
- Can be installed from the browser to the home screen or desktop
- Feels like an app after installation
- Uses the camera for barcode and QR scanning
- Supports NFC on compatible devices
- Adapts to mobile and larger screens

### Installation for end users
In compatible browsers, the app can be saved using options such as "Install" or "Add to Home Screen". After that, paw.oxs.at can be opened directly from the device without entering the address each time.

### Important usage notes
- Camera and, where available, NFC permissions must be allowed for scanning features
- Not every device supports NFC in the browser
- Without signing in, only public shared content is visible
- Advanced features depend on the account, role, and sharing settings

## In short

The PWA on paw.oxs.at is the central user interface of the project. It combines animal management, scan features, digital documents, and controlled sharing in an installable web app that works on mobile devices and desktop systems.
